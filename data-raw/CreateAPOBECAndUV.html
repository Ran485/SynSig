Overview and rationale

Checking for interference between C > T mutation in APOBEC (SBS2) and UV (SBS7a,b).

Required libraries

library(ICAMS)
library(SynSig)

Data set 1, bladder and skin melanoma to explore SBS2 and SBS7a,b

500 synthetic bladder transitional cell carcinomas (high in SBS2 and SBS13) and 500 synthetic skin melanomas (high in SBS7a,b,c,d). The potential interference is between SBS2 (mainly C > T) and SBS7a,b (mainly C > T).

CopyScripts <- function() {
  res <- file.copy(from = "1run.signatureanalyzer.on.monster.R",
                   to = OutDir("sa.sa.96"))
  if (!res) cat("Copy failed\n")
  res <- file.copy(from = "2run.signatureanalyzer.on.monster.R",
                   to = OutDir("sp.sp/"))
  if (!res) cat("Copy failed\n")
  res <- file.copy(from = "3run.signatureanalyzer.on.monster.R",
                   to = OutDir("sa.sa.COMPOSITE/"))
  if (!res) cat("Copy failed\n")
  res <- file.copy(from = "4run.signatureanalyzer.on.monster.R",
                   to = OutDir("sp.sa.COMPOSITE/"))
  if (!res) cat("Copy failed\n")
